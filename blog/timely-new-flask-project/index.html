Hello there!

Today I’d like to present you my new Flask project called Timely. It came from not so popular apps to countdown time. I was searching the solution for myself to use it on mobile, desktop and tablet. I haven’t found such a good product so I make an own little app to help me with measuring my focus time.

If you want to examine how it’s working and looking - check out the live demo of an app deployed on Google Cloud Platform.

For more info about code and stack see the Github repository.

I hope you will find the app as a practical thing in your everyday life.
